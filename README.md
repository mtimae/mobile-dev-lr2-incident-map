# ЛР2 — Карта происшествий

## Описание проекта
Мобильное Android-приложение **«Карта происшествий»**, разработанное в рамках лабораторной работы по дисциплине **«Разработка мобильных приложений»**.

Приложение предназначено для отображения карты, определения текущего местоположения пользователя и добавления на карту точек происшествий двух типов:
- ДТП;
- ремонт дороги.

## Тема лабораторной работы
**Работа с географическими информационными системами**

## Вариант
**Вариант 13 — «Карта происшествий»**

## Используемые технологии
- Kotlin
- Android Studio
- XML-разметка интерфейса
- osmdroid (OpenStreetMap)
- Fused Location Provider
- SharedPreferences

## Функциональные возможности
- отображение карты на экране смартфона;
- запрос разрешения на доступ к геолокации;
- определение текущего местоположения пользователя;
- центрирование карты по координатам пользователя;
- добавление точек происшествий на карту;
- выбор типа происшествия:
  - ДТП;
  - ремонт дороги;
- сохранение добавленных точек и их восстановление после повторного запуска приложения.

## Структура проекта
- `MainActivity.kt` — основная логика приложения;
- `activity_main.xml` — главный экран приложения;
- `AndroidManifest.xml` — разрешения и конфигурация приложения;
- `strings.xml` — строковые ресурсы интерфейса.

## Инструкция по запуску
1. Установить Android Studio.
2. Открыть проект.
3. Подключить Android-смартфон по USB.
4. Включить режим разработчика и USB-отладку на телефоне.
5. Выбрать устройство в Android Studio.
6. Нажать **Run**.

## Инструкция по использованию
1. Запустить приложение на смартфоне.
2. Разрешить доступ к местоположению.
3. Нажать кнопку **«Моё местоположение»** для определения текущих координат.
4. Нажать кнопку **«Добавить точку»**.
5. Выбрать тип происшествия:
   - ДТП;
   - ремонт дороги.
6. Проверить отображение маркера на карте.

## Результат
Разработано мобильное приложение, отображающее карту, текущее местоположение пользователя и точки происшествий различных типов с возможностью сохранения данных между запусками.
